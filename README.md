# EyeTalk2U (Vite + React + TypeScript)

This project is a **web-based eye-tracking interface** built using **Vite**, **React**, and **TypeScript**.  
It provides a modular UI structure and efficient browser GUI for EyeTalk2U (e.g., webcam access, video frame analysis, pupil detection, etc.).
